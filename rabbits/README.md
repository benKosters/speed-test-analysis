**RABBITS Toolkit Information**

*This sub-directory contains the files required for filtering out the relevant netlog events/data. The toolkit itself is located in another repository, CAIDA/rabbits-perf*

1) filter-rabbits-netlog-data.js

Main script for filtering out relevant data such as raw byte counts, timestamps, HTTP stream IDs, and socket IDs.

2) file-utils.js

Utility functions for reading/writing to files and other functions to help standardize the RABBITS toolkit output files with the Ookla Conventional test output files.
